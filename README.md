# Online Shopping App

It is a JDBC JAVA project with customers, admin and suppliers. It allows customers to buy the products online and suppliers to sell their products online.

Requirements :

openjdk==17.0.2\
JUnit==5\
mysql==8.0.23\
mysql-connector-java-5.1.49.jar
